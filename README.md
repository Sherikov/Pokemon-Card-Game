# Pokemon Memory Game

This project was created as practice for working with React hooks, especially `useState` and `useEffect`.
The main goal was to build a small interactive memory game that loads Pokemon data from PokeAPI, displays a 3x3 card board, tracks the current score, and reshuffles the cards after each click.

This project is not fully handmade.
Some parts were created with AI assistance.
In particular, the general project mockup and visual layout were developed by combining my own ideas with AI help to turn those ideas into a working design.

## Tech Stack

- React
- Vite
- CSS
- PokeAPI

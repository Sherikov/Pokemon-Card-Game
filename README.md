# Pokemon Memory Game

This project was created as practice for working with React hooks, especially `useState` and `useEffect`. The main goal was to build a small interactive memory game that loads Pokemon data from PokeAPI, displays a 3x3 card board, tracks the current score, and reshuffles the cards after each click.

> **Note:** This project is not fully handmade. Some parts were created with AI assistance. In particular, the general project mockup and visual layout were developed by combining my own ideas with AI help to turn those ideas into a working design.

## Tech Stack
* **React** — UI library
* **Vite** — Build tool
* **CSS** — Styling
* **PokeAPI** — Fetching Pokemon data (images and names)

## Features
* Asynchronous data fetching via a third-party API.
* Tracking the current score and saving the highest score (Best Score).
* Random card shuffling algorithm triggered on every interaction.
* Responsive design for comfortable gameplay.

## Installation and Local Setup

1. Clone the repository:
   ```bash
   git clone [https://github.com/Sherikov/Pokemon-Card-Game.git](https://github.com/Sherikov/Pokemon-Card-Game.git)
   ```
2. Navigate to the project directory:
  ```Bash
  cd Pokemon-Card-Game
  ```
3. Install dependencies:
  ```Bash
    npm install
```
4. Start the development server:
  ```Bash
  npm run dev
  ```
5. Deployment
The project is live and can be played here: Pokemon Memory Game on GitHub Pages
